## Quick Start

### Option 1: Local development

```bash
./start.sh
```

Or manually:

```bash
npm install
cd go && go build -o ../bin/server ./cmd/server && cd ..
cp .env.example .env   # edit with your values
npm run dev
```

### Option 2: Docker

```bash
cp .env.example .env   # edit with your values
npm run docker:dev
```
